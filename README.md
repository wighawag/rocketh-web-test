# Reproduction

1. clone (with git submodule)

```bash
git clone --recurse-submodules https://github.com/wighawag/rocketh-web-test
```

2. install deps

```bash
pnpm i
```

3. compile and export deploy scripts:

```bash
pnpm compile
```

4. run server

```bash
pnpm dev
```

5. navigate to [http://localhost:5173](http://localhost:5173)

6. click the button "test" and check browser log
